# SecondTeamProject

------------
> ##프로젝트 소개
#### 프로젝트명 : 쌍용인력센터
#### 중략 : Spring 기반의 잡 플랫폼 사이트 구현

### 주요 관리자 주요 기능
#### 1. 회원가입/ 이력서 작성 / 이력서 출력
#### 2. 게시판 글 추가,삭제,수정,댓글,대댓글,스크랩
#### 3. 잡 지식인 게시판 답변 채택시 내공 부여(Admin)
#### 4. 채용달력 날짜 선택 시 마감 별 공고 출력
